# Indovina il numero

Un gioco semplicissimo realizzato in Python.

## Come funziona?

Il computer pensa a un numero segreto tra 1 e 20. Tu devi indovinare quale numero ha scelto. Dopo ogni tentativo, il computer ti dice se hai sparato troppo alto o troppo basso.

## Cosa impari con questo progetto?

- Usare `print()` per mostrare messaggi
- Chiedere input all'utente con `input()`
- Convertire testo in numero con `int()`
- Usare `if`, `elif`, `else` per prendere decisioni
- Creare un ciclo `while True` per ripetere qualcosa fino a quando non indovini
- Generare numeri casuali con `random.randint()`

## Come eseguirlo?

1. Assicurati di avere Python installato (scaricabile da python.org)
2. Salva il file come `indovina_numero.py`
3. Apri il terminale (o prompt dei comandi) nella cartella dove hai salvato il file
4. Scrivi: `python indovina_numero.py`
5. Gioca e divertiti!
